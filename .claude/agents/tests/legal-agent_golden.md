# Золотые тест-кейсы: legal-agent

Используются при обновлении промпта субагента для проверки, что качество не ухудшилось.

---

## Тест 1: Анализ договора

**Вход:** «Разбери договор с CAD-разработчиком. Формат — таблица обязательств.»

**Ожидаемый результат:**
- Каждое обязательство с точным номером пункта: «п. 3.2 Договора»
- Ссылка на конкретный файл: documents/contracts/Договор_RedPeak_v3_2_с_Cad_разработчиком_v3_FINAL.pdf
- Разделение обязательств по сторонам (Заказчик / Исполнитель)
- Указаны сроки и суммы, если есть в документе

**Красные флаги:**
- Номера пунктов, которых нет в реальном документе
- Отсутствие ссылки на файл
- Выдуманные статьи закона
- Утверждения о содержании договора без проверки конкретного пункта

---

## Тест 2: Анализ рисков по договору

**Вход:** «Какие юридические риски в NDA с АСКОН? Формат — записка.»

**Ожидаемый результат:**
- Каждый риск привязан к конкретному пункту NDA (с номером)
- Указан источник файла
- Разграничены: факты из документа vs [МОЯ ИНТЕРПРЕТАЦИЯ]
- Ссылки на нормы закона верифицированы через веб-поиск или помечены [ПРОВЕРИТЬ НОРМУ]

**Красные флаги:**
- Ссылки на конкретные статьи ГК/НК без верификации
- Пункты договора, которых в документе нет
- Абстрактные риски без привязки к тексту («возможны разногласия» -- по какому пункту?)

---

## Тест 3: Анализ NDA с ML-инженером

**Вход:** «Разбери NDA с ML-инженером Кадыровым из DATA/NDA/»

**Ожидаемое поведение:**
- Загружает конкретный файл NDA из DATA/NDA/
- Таблица обязательств обеих сторон
- Срок действия NDA
- Пункты об IP и конфиденциальности
- Штрафные санкции с точными суммами из документа

**Красные флаги:**
- Суммы штрафов не из документа
- Пункты не существуют в NDA
- Перепутаны стороны (кто на кого накладывает обязательства)

---

## Тест 4: Сравнение двух редакций договора

**Вход:** «Сравни v2 и v3 договора с CAD-разработчиком.»

**Ожидаемое поведение:**
- Загружает оба файла из documents/contracts/ или DATA/Договоры/
- Таблица различий: пункт | было (v2) | стало (v3) | комментарий
- Привязка к конкретным номерам пунктов в обоих файлах
- Оценка рисков изменений

**Красные флаги:**
- Различия не привязаны к пунктам
- Цитаты не из реальных документов
- Пропущены существенные изменения
- Нет ссылок на оба файла
