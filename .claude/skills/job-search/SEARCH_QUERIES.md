# Поисковые запросы для hh.ru

Ротировать запросы при каждом запуске. Все запросы с фильтрами: удаленно.

## Основные (использовать всегда)
1. `инженер конструктор` (schedule=remote, work_format=REMOTE)
2. `реверс инжиниринг` (schedule=remote, work_format=REMOTE)

## Дополнительные (ротация, 2-3 за запуск)
3. `инженер КОМПАС-3D`
4. `инженер SolidWorks удаленно`
5. `3D сканирование инженер`
6. `конструктор нефтегазовое оборудование`
7. `инженер проектировщик машиностроение`
8. `Geomagic инженер`
9. `конструктор импортозамещение`

## Формат URL
```
https://hh.ru/search/vacancy?text={запрос}&schedule=remote&work_format=REMOTE&enable_snippets=true&ored_clusters=true&order_by=relevance
```
