# Скилл: Поиск вакансий на hh.ru

## Входные данные
- Резюме кандидата: `DATA/Партфолио/Чадаев Владислав Васильевич резюме.pdf`
- Лог вакансий: `DATA/job_search/vacancies_log.jsonl`
- Сопроводительные письма: `DATA/job_search/cover_letters_*.md`

## Алгоритм

### 1. Поиск вакансий
Открыть hh.ru через Claude in Chrome и выполнить поиск по каждому запросу из SEARCH_QUERIES.md.
Фильтры: удаленно (schedule=remote, work_format=REMOTE).

### 2. Сбор данных
Для каждой найденной вакансии собрать:
- Название, компания, зарплата, формат работы
- Требования и обязанности
- URL вакансии

### 3. Дедупликация
Проверить `DATA/job_search/vacancies_log.jsonl` - не откликались ли уже на эту вакансию.

### 4. Скоринг
Оценить каждую вакансию по критериям из SCORING.md. Порог: >= 60 баллов.

### 5. Генерация сопроводительного письма
Для вакансий выше порога - сгенерировать персонализированное письмо по шаблону из TEMPLATE_COVER_LETTER.md.

**КРИТИЧЕСКИ ВАЖНО:** Использовать ТОЛЬКО факты из резюме. Не приписывать навыки, не менять формулировки. Перед финализацией - проверить каждое утверждение по тексту резюме.

### 6. Подтверждение руководителя
Показать список вакансий с письмами. Ждать подтверждения: какие отправляем.

### 7. Отклик
Через Claude in Chrome:
1. Открыть вакансию на hh.ru
2. Нажать "Откликнуться"
3. Выбрать резюме "Инженер-конструктор, 180 000 ₽"
4. Добавить сопроводительное письмо
5. Нажать "Откликнуться" (submit в модале через find + ref)
6. Проверить "Резюме доставлено"

### 8. Логирование
Записать результат в `DATA/job_search/vacancies_log.jsonl`.

## Файлы скилла
- `SKILL.md` - этот файл (точка входа)
- `SEARCH_QUERIES.md` - поисковые запросы
- `SCORING.md` - критерии скоринга
- `TEMPLATE_COVER_LETTER.md` - шаблон сопроводительного
