# Критерии скоринга вакансий

Оценить каждую вакансию по критериям ниже. Сумма >= 60 - кандидат для отклика.

| Критерий | Баллы | Пояснение |
|---|---|---|
| КОМПАС-3D или SolidWorks в требованиях | +25 | Основные инструменты кандидата |
| Реверс-инжиниринг или 3D-сканирование | +25 | Ключевая компетенция |
| Нефтегаз или машиностроение | +15 | Отраслевой опыт |
| Санкт-Петербург, офис или гибрид | +15 | Обязательное условие |
| ЕСКД / КД в требованиях | +10 | Подтвержденный навык |
| Зарплата >= 180к | +5 | Целевой диапазон |
| Зарплата >= 200к | +5 | Выше целевого |

## Дисквалификация (не откликаться)
- Только удалённо (без офиса/гибрида в СПб)
- Не Санкт-Петербург (Москва, другие регионы — только если явно указан СПб-офис)
- Зарплата ниже 150к (если указана явно)
- Требуется ПО, которого нет в резюме, без возможности обучения (NX, CATIA как основные)
- Мебельное производство (Базис-мебельщик)
- Студенческие работы / фриланс-биржи
- Уже откликались (проверить vacancies_log.jsonl)
