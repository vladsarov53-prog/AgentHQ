# Шаблон сопроводительного письма

## Структура (5-7 предложений)

1. **Приветствие:** "Здравствуйте!"
2. **Представление** (1 предложение): кто я, ключевой опыт, совпадающий с вакансией
3. **Релевантный опыт** (2-3 предложения): конкретные достижения из резюме, которые совпадают с требованиями вакансии
4. **Закрытие гэпов** (если есть): "Готов быстро освоить [навык]"
5. **Завершение:** "Буду рад обсудить."

## Правила

- Тон: деловой, конкретный, без канцелярита
- ТОЛЬКО факты из резюме. Ни одного выдуманного навыка или достижения
- Не писать "готов взять на себя проверку" и подобные обещания, не подтвержденные опытом
- Цифры из резюме: 50+ узлов, 700+ деталей, 15 инженеров обучено, 2-3 недели на узел
- Не использовать длинное тире
- Не начинать предложения с "Важно отметить" и подобных вводных
