# Theme Factory

Набор профессиональных тем для стилизации презентаций, документов, отчётов и HTML-страниц.

## 10 готовых тем

### 1. Ocean Depths
- Палитра: глубокий синий (#1a365d), бирюза (#2b6cb0), белый (#ffffff)
- Шрифты: Montserrat (заголовки), Source Sans Pro (текст)
- Характер: профессиональный, спокойный, доверительный

### 2. Modern Minimalist
- Палитра: чёрный (#1a1a1a), белый (#fafafa), серый (#6b7280), акцент (#3b82f6)
- Шрифты: Inter (заголовки), System UI (текст)
- Характер: чистый, современный, технологичный

### 3. Forest Canopy
- Палитра: тёмно-зелёный (#1a4731), салатовый (#68d391), кремовый (#fffff0)
- Шрифты: Playfair Display (заголовки), Lato (текст)
- Характер: экологичный, натуральный, премиальный

### 4. Sunset Boulevard
- Палитра: тёплый оранжевый (#dd6b20), розовый (#ed64a6), тёмный (#2d3748)
- Шрифты: Poppins (заголовки), Open Sans (текст)
- Характер: энергичный, креативный, смелый

### 5. Golden Hour
- Палитра: золотой (#d69e2e), тёмно-коричневый (#744210), кремовый (#fffbeb)
- Шрифты: Cormorant Garamond (заголовки), Nunito (текст)
- Характер: элегантный, premium, инвестиционный

### 6. Arctic Frost
- Палитра: ледяной голубой (#e2e8f0), белый (#ffffff), тёмно-серый (#2d3748)
- Шрифты: Raleway (заголовки), Roboto (текст)
- Характер: минималистичный, холодный, технический

### 7. Desert Rose
- Палитра: пыльный розовый (#feb2b2), терракота (#c53030), песочный (#fefcbf)
- Шрифты: DM Serif Display (заголовки), DM Sans (текст)
- Характер: тёплый, утончённый, артистичный

### 8. Tech Innovation
- Палитра: электрический синий (#4299e1), неон-зелёный (#48bb78), тёмный (#1a202c)
- Шрифты: Space Grotesk (заголовки), IBM Plex Sans (текст)
- Характер: инновационный, tech-forward, стартап

### 9. Botanical Garden
- Палитра: изумрудный (#276749), лавандовый (#9f7aea), слоновая кость (#fffff0)
- Шрифты: Fraunces (заголовки), Work Sans (текст)
- Характер: органичный, изысканный, научный

### 10. Midnight Galaxy
- Палитра: тёмно-фиолетовый (#2d1b69), звёздный (#a78bfa), серебро (#e2e8f0)
- Шрифты: Outfit (заголовки), Inter (текст)
- Характер: футуристичный, премиальный, визионерский

## Процесс применения

1. Выбрать тему (или описать желаемый стиль для генерации новой)
2. Применить цвета и шрифты к артефакту
3. Обеспечить контраст и читаемость
4. Проверить единообразие на всех слайдах/страницах

## Генерация новой темы

Если ни одна тема не подходит, опиши:
- Настроение / характер
- Целевая аудитория
- Контекст использования (инвесторы, ФСИ, команда, конференция)

Система создаст уникальную тему с палитрой и шрифтами.
